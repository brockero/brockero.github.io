CURRENT - Design, Front-End Development, and Deployment

This portfolio originated as a class assignment for WRA 210 - Introduction to Web Authoring. It involved working through three phases and ended with a presentation of my work to my peers.

The first phase was creating an information architecture and wireframing the website. I was tasked with creating both a site map and a navigation. I also had to wireframe each page that I had within my navigation.

The second phase was design. Part of the requirements were that we create design documentation for the entire website. This included creating multiple color palettes, a typographical reference card, and full design docs for each page of our website.

Finally, I had to build my portfolio and present it to the class. At the time of its inital development, many differences grew out from the original documentation. Other pages I hadn't even though about before were created to fit needs that arose. Each page was hand-coded by me, but I also used the Bootstrap framework for layout and structure. It is entirely HTML5 and CSS3 standards compliant and is (in my eyes) quite beautiful for an inital portfolio.

Throughout design and development, I learned a lot about the whole process. It was hard being my own client, since I wanted to jump right into development, but I stuck through the proper pacing and development cycle to make sure that I didn't miss anything. A lot of work went into building this and there were a lot of stressful days staring at one problem for hours on end, but through it all I became a better, more knowledgeable designer and web developer.
